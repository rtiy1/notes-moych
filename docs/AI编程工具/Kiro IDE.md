# Kiro IDE

官网：https://kiro.dev/

亚马逊开发的AI IDE编程工具，只提供claude模型。

项目从0到1体验很好，spec模式让人眼前一亮，agent写完task后用户可以自由选择执行。而且，新用户可以享受550额度，爽用opus 4.5，并且kiro可以用临时邮箱无限注册。不过项目构建完毕后，修改代码能力较差，容易打转半天解决不了问题。评价为顶级。但是利用反代理到Claude CLI使用Opus4.5，直接夯。

##### cloudflare域名邮箱可以无限注册kiro（目前封号风险较高）

##### 可以在某宝购买163邮箱注册，1块钱30个非常便宜，注册github，利用github注册kiro即可。
