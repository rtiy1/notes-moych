## 💡 学习建议

Python 与 Java 在设计哲学上有很大不同：
- **简洁性**：Python 强调“用一种方法做一件事”。
- **动态性**：类型是在运行时确定的。
- **强制缩进**：初学者最容易在缩进上出错，请务必保持一致。
