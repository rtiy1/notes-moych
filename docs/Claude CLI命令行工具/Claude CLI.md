# Claude CLI

目前用过的最顶级的编程工具，代码准确率高，agent也非常智能。

目前我的项目大多数时间都是使用claude，用它写PRD，写技术文档非常顶级，写完之后如果为了节省用量可以把写好的文档喂给其他AI进行编写。

Claude支持第三方API接入，claude模型本身太贵了，如果预算吃紧，国产模型智谱GLM有claude套餐，20块钱一个月，非常好用，用起来也有claude模型的80%的能力。

只能说，强的不仅仅是claude的模型，它的agent也非常强，用其他模型接入实力也很强，评价为夯爆了。

目前kiro可以利用反代理将claude模型转为API，在claude cli里通过API接入，享受满血claude。
