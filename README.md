# 溟如池鱼的个人笔记

基于 VitePress 搭建的个人学习笔记网站。

## 技术栈

- VitePress ^1.6.4
- Element Plus ^2.13.0
- Sass ^1.97.1

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build
```

## 目录结构

```
docs/
├── 算法笔记/    # 滑动窗口、链表等算法
├── 后端笔记/    # SSM、Redis 等后端技术
└── 其他笔记/    # Git、Bug记录等
```
